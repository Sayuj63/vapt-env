================================================================================
ACTION FORMAT
================================================================================

The environment expects actions in this format:

{
  "action_type": "list_tools" | "use_tool" | "submit_finding" | "generate_report",
  "tool_name": "string" (required for use_tool),
  "arguments": {
    "host": "string",
    "title": "string",
    "severity": "low" | "medium" | "high",
    ...
  }
}

Examples:

1. List Tools:
   {"action_type": "list_tools"}

2. Use Tool:
   {
     "action_type": "use_tool",
     "tool_name": "network_scan",
     "arguments": {"host": "192.168.1.1"}
   }

3. Submit Finding:
   {
     "action_type": "submit_finding",
     "arguments": {
       "title": "SQL Injection",
       "host": "192.168.1.1",
       "severity": "high",
       "cvss_score": 9.8,
       "cwe": "CWE-89",
       "owasp": "A03:2021"
     }
   }

4. Generate Report:
   {"action_type": "generate_report"}

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Connection refused
Solution: Check HF Space status, wait 30 seconds, retry

Issue: CUDA out of memory
Solution: Use CPU or smaller model

Issue: HF_TOKEN not found
Solution: Add to Colab Secrets (🔑 icon)

Issue: Model download fails
Solution: Check internet, verify HF_TOKEN, retry

Issue: Low training reward
Solution: Try more episodes or different scenario

See COLAB_TRAINING_README.md for detailed troubleshooting.
