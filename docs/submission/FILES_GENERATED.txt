TROUBLESHOOTING QUICK REFERENCE
═════════════════════════════════════════════════════════════════════════════

Issue: Connection refused
→ Check HF Space status, wait 30 seconds, retry

Issue: CUDA out of memory
→ Use CPU or smaller model

Issue: HF_TOKEN not found
→ Add to Colab Secrets (🔑 icon)

Issue: Model download fails
→ Check internet, verify HF_TOKEN, retry

Issue: Low training reward
→ Try more episodes or different scenario

For more help: See COLAB_TRAINING_README.md
